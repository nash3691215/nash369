Introduction

Ce que ce guide permet vraiment

Ce guide te donne une méthode concrète pour créer un site web fonctionnel en utilisant l'intelligence artificielle (Claude) et un hébergeur simple (Hostinger). Pas de code à apprendre. Pas de logiciel complexe à maîtriser.

Avant l'IA, créer un site professionnel signifiait soit payer un développeur (1000€ à 5000€), soit passer des semaines à apprendre WordPress ou un autre outil. Cette époque est en train de changer.

Soyons clairs sur les délais

Ce que "5 minutes" signifie réellement

En 5 minutes, tu peux générer le code d'un site complet et le mettre en ligne. C'est le temps technique minimum.

En 30 à 60 minutes, tu auras un site vraiment professionnel : avec tes vraies images, tes textes personnalisés, un formulaire fonctionnel et les ajustements visuels nécessaires.

C'est normal. La génération est rapide, la personnalisation prend du temps. Ce guide couvre les deux.

Ce que ce guide te permettra de faire

  • Générer la structure complète d'un site en quelques minutes
  • Mettre ce site en ligne sur ton propre domaine
  • Personnaliser le design, les textes et les images
  • Rendre le site responsive (adapté mobile)
  • Connecter un formulaire de contact fonctionnel
  • Appliquer les bases du SEO

Ce que ce guide ne promet pas

Ce guide ne fera pas de toi un développeur web. Il ne remplace pas un travail de branding approfondi. Il ne couvre pas les sites e-commerce complexes ni les applications web sur mesure.

L'objectif est simple : te donner l'autonomie pour créer des sites vitrines, landing pages et portfolios rapidement, sans dépendre de personne pour chaque modification.

Chapitre 1 — La Méthode Express

Cette méthode se décompose en trois étapes. Une fois maîtrisée, tu pourras la répéter pour n'importe quel type de site.

1

2

Étape 1 : Le brief

Avant de parler à Claude, définis ce que tu veux. Plus ton brief est précis, meilleur sera le résultat. Utilise cette structure :

Structure du brief
  • Type de site : vitrine, landing page, portfolio, événement
  • Activité : ce que tu fais, pour qui
  • Sections : les blocs que tu veux (hero, services, contact...)
  • Couleurs : couleur principale + secondaire
  • Ambiance : minimaliste, chaleureux, corporate, dynamique
  • Action principale : ce que le visiteur doit faire (appeler, réserver, s'inscrire)

Étape 2 : La génération avec Claude

Voici le prompt universel. Copie-le et adapte les éléments entre crochets :

Prompt à copier Crée un site web one-page complet en HTML/CSS pour [ACTIVITÉ]. SECTIONS : - Header avec logo texte et navigation - Hero section avec titre accrocheur et bouton d'action - Section [À PROPOS / SERVICES / OFFRES] - [AUTRES SECTIONS selon besoin] - Section contact avec formulaire - Footer avec mentions légales STYLE : - Design moderne et professionnel - Couleur principale : [COULEUR] - Couleur secondaire : [COULEUR] - Ambiance : [STYLE] - Responsive mobile-first CONTRAINTES : - Un seul fichier HTML avec CSS intégré - Pas de dépendances externes - Images en placeholder (je les remplacerai) Génère le code complet.

Exemple concret

Exemple — Restaurant Crée un site web one-page complet en HTML/CSS pour un restaurant italien "La Dolce Vita" à Lyon. SECTIONS : - Header avec logo texte et navigation - Hero section avec image de fond et slogan - Notre histoire (2 paragraphes) - Menu avec 3 catégories (entrées, plats, desserts) - Horaires et localisation - Formulaire de réservation - Footer STYLE : - Design élégant et chaleureux - Couleur principale : bordeaux (#722F37) - Couleur secondaire : crème (#F5F5DC) - Ambiance : trattoria italienne authentique - Responsive mobile-first CONTRAINTES : - Un seul fichier HTML avec CSS intégré - Images en placeholder Génère le code complet.

Étape 3 : La mise en ligne sur Hostinger

Une fois le code généré par Claude :

  1. Copie le code — Clique sur "Copier" dans Claude, ouvre un éditeur de texte (Notepad, TextEdit), colle le code, enregistre sous index.html
  2. Teste localement — Double-clique sur le fichier pour l'ouvrir dans ton navigateur. Vérifie que tout s'affiche correctement.
  3. Connecte-toi à Hostinger — Va dans hPanel → Fichiers → Gestionnaire de fichiers → dossier public_html
  4. Upload le fichier — Clique sur "Télécharger", sélectionne ton index.html, attends quelques secondes
  5. Active le SSL — Dans hPanel → Sécurité → SSL → Installer Let's Encrypt (gratuit)
Ton site est en ligne Accessible sur ton-domaine.com

Modifier le site après génération

Claude conserve le contexte de la conversation. Tu peux lui demander des modifications sans tout recommencer :

Modifications Change la couleur principale en bleu marine (#1a365d)
Modifications Ajoute une section témoignages avec 3 avis clients
Modifications Rends le header fixe en haut de page au scroll
Conseil pratique

Chaque modification génère une nouvelle version complète du code. Copie et remplace ton fichier index.html à chaque fois, puis re-upload sur Hostinger.

Professionnalisation

Chapitre 2 — Rendre le Site Professionnel

Un site généré en 5 minutes fonctionne, mais il contient des placeholders et des réglages par défaut. Cette étape transforme le brouillon en site crédible.

1. Remplacer les images placeholder

Claude génère des rectangles gris ou des liens vers des images génériques. Tu dois les remplacer par de vraies images.

Source Type de contenu Licence
unsplash.com Photos haute qualité Gratuit, sans attribution
pexels.com Photos et vidéos Gratuit, sans attribution
undraw.co Illustrations SVG Gratuit, personnalisable
Intégrer une image Remplace l'image placeholder du hero par cette URL : https://images.unsplash.com/photo-xxxxx
Optimisation des images

Compresse tes images avant upload avec tinypng.com. Une image de 3 Mo ralentit le site. Vise moins de 200 Ko par image.

2. Vérifier le responsive (mobile)

Claude génère du code responsive par défaut, mais vérifie toujours le rendu sur mobile :

  1. Ouvre Chrome → clic droit → Inspecter (ou F12)
  2. Clique sur l'icône mobile/tablette en haut à gauche des DevTools
  3. Teste sur "iPhone SE" et "iPad" pour vérifier l'affichage
  4. Si un élément déborde ou s'affiche mal, décris le problème à Claude
Correction mobile Sur mobile, le menu déborde de l'écran. Ajoute un menu hamburger qui s'ouvre au clic.

3. Connecter un formulaire fonctionnel

Par défaut, un formulaire HTML ne fait rien. Pour qu'il envoie vraiment les données, utilise un service gratuit comme Formspree :

  1. Crée un compte sur formspree.io (gratuit jusqu'à 50 soumissions/mois)
  2. Crée un nouveau formulaire et copie l'ID fourni
  3. Demande à Claude d'intégrer cet ID dans ton formulaire
Formspree Modifie le formulaire de contact pour utiliser Formspree. L'action doit être : https://formspree.io/f/TON_ID La méthode : POST

4. Appliquer les bases du SEO

Le SEO (référencement) permet à ton site d'apparaître dans Google. Voici les éléments essentiels :

SEO de base Optimise le SEO de ce site : - Title : [Nom] — [Activité] à [Ville] - Meta description : [Description de 150-160 caractères] - Ajoute les balises Open Graph pour le partage social - Vérifie que toutes les images ont des attributs alt descriptifs

Les éléments SEO essentiels

Élément Rôle Exemple
Title Titre dans l'onglet et Google La Dolce Vita — Restaurant Italien Lyon
Meta description Description sous le lien Google Restaurant italien authentique au cœur de Lyon. Pâtes fraîches, pizzas au feu de bois. Réservation en ligne.
Alt images Description des images pour Google Salle du restaurant La Dolce Vita avec tables dressées

5. Personnaliser les textes

Claude génère des textes génériques. Remplace-les par tes propres contenus. Sois concret et spécifique :

Évite
"Nous offrons des services de qualité à nos clients depuis de nombreuses années."
Préfère
"Depuis 2018, nous accompagnons 150+ PME dans leur transformation digitale."

Checklist — Site professionnel

  • Images placeholder remplacées par de vraies photos
  • Images compressées (moins de 200 Ko chacune)
  • Responsive vérifié sur mobile et tablette
  • Formulaire connecté à Formspree (ou équivalent)
  • Title et meta description personnalisés
  • Textes génériques remplacés par ton contenu
  • Coordonnées et mentions légales à jour
  • SSL activé (cadenas HTTPS visible)

Optionnel

Chapitre 4 — Optimisations Avancées

Ce chapitre est optionnel. Les optimisations suivantes ne sont pas nécessaires pour avoir un site fonctionnel, mais elles peuvent améliorer l'expérience utilisateur et les performances.

Ajouter Google Analytics

Google Analytics te permet de voir combien de personnes visitent ton site et d'où elles viennent.

  1. Crée un compte sur analytics.google.com
  2. Ajoute une propriété (ton site)
  3. Récupère l'ID de mesure (format G-XXXXXXXXXX)
  4. Demande à Claude de l'intégrer
Analytics Ajoute le code Google Analytics 4 dans le head. Mon ID de mesure : G-XXXXXXXXXX

Améliorer les performances

Performances Optimise ce site pour la performance : - Ajoute loading="lazy" sur toutes les images - Minifie le CSS - Ajoute les attributs width et height sur les images

Personnalisations visuelles

Quelques améliorations visuelles que tu peux demander :

Mode sombre Ajoute un toggle mode clair/sombre dans le header
WhatsApp Ajoute un bouton WhatsApp flottant en bas à droite. Numéro : +33 6 XX XX XX XX
Animations Ajoute des animations fade-in sur les sections au scroll

Intégrer une carte Google Maps

  1. Va sur Google Maps et recherche ton adresse
  2. Clique sur Partager → Intégrer une carte
  3. Copie le code iframe
  4. Envoie-le à Claude pour l'intégrer
Note sur les optimisations

Ces améliorations sont utiles mais secondaires. Un site simple et rapide vaut mieux qu'un site complexe et lent. Ajoute ces éléments uniquement si tu en as vraiment besoin.
